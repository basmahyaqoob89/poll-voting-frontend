 Poll Voting Platform

A simple poll voting platform built with React and Vite.

Users can:
- Create polls
- Vote in real time
- View live results
- Share polls using QR codes
- Send results via email

---

# 🚀 Technologies Used

- React
- Vite
- React Router DOM
- AWS API Gateway
- AWS Lambda
- QRCode React

---

# 📦 Setup Instructions

## Clone the repository

```bash
git clone YOUR_GITHUB_REPOSITORY_LINK
```

## Open the project

```bash
cd poll-frontend
```

## Install dependencies

```bash
npm install
```

## Run the project

```bash
npm run dev
```

Open:

```bash
http://localhost:5173
```
